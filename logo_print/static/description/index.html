Overview

This module enables us to store the chosen submitted logos by customers in a stand-alone model.


In addition to that, the administrator can upload the collection of logo positions on the website.

Features

Creating a library of collections from which customers choose logo positions
Creating a library of logo submissions by customers which have unique indexed names
Inherithing logo submissions in sales orders & sales orders lines
Customizing shop website template and making it a user friendly interface by using a dialog window
Customizing add to cart template to make uploading logos for specific positions easy and accurate
